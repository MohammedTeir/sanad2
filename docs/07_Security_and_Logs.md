# وثيقة التصميم الشاملة لنظام إدارة المخيمات
## الجزء السابع: الأمن والسجلات

## 10) أمن البيانات وحمايتها

### 10.1 التشفير
- **أثناء النقل:** HTTPS / SSL.
- **أثناء التخزين:** تشفير قاعدة البيانات، وتشفير الحقول الحساسة (AES-256).

### 10.2 سجل التدقيق (Audit Log)
- تسجيل شامل لعمليات البيانات (إضافة، تعديل، حذف منطقي).
- يسجل: من قام بالعملية، التاريخ، القيمة القديمة والجديدة.
- **لا يمكن حذفه** أو تعديله.

### 10.3 الحذف المنطقي (Soft Delete)
- البيانات لا تحذف نهائياً بل توسم بـ `is_deleted = true`.
- يمكن استعادتها، وتستثنى من نتائج البحث العادية.

### 10.4 الأمان ضد الهجمات
- حماية ضد SQL Injection, XSS, CSRF.
- حماية ضد Brute Force (تحديد محاولات الدخول).

## 15) سجل عمليات النظام (System Operations Log)

### 15.1 الوصف
سجل تقني منفصل عن سجل تدقيق المستخدمين.

### 15.2 ما يتم تسجيله
- عمليات المزامنة (نجاح/فشل، تعارضات).
- عمليات النسخ الاحتياطي والاستعادة.
- محاولات الدخول (الناجحة والفاشلة).
- الأخطاء البرمجية (Errors) والتنبيهات الأمنية.

### 15.3 الاحتفاظ والأرشفة
- يُحفظ لمدة 12 شهراً ثم يؤرشف تلقائياً.
