# سند - منظومة إدارة مخيمات غزة

نظام متكامل لإدارة مخيمات النازحين في غزة، مع التركيز على تتبع الأفراد واحتياجاتهم وتقديم المساعدات بشكل فعال.

## الميزات

- تتبع الأفراد والمستفيدين
- إدارة المخيمات والسكان
- إدارة توزيع المساعدات
- تتبع المخزون
- تقييم درجات الهشاشة باستخدام الذكاء الاصطناعي
- أنظمة الأمان المتقدمة

## الإعداد

**المتطلبات:** Node.js

1. تثبيت الحزم:
   ```bash
   npm install
   ```

2. إعداد المتغيرات البيئية:
   - انسخ `.env.example` إلى `.env`
   - أدخل مفاتيح API الخاصة بك
   - أدخل بيانات اعتماد Supabase

3. إعداد قاعدة البيانات:
   ```bash
   npm run setup:db
   ```

4. تشغيل التطبيق:
   ```bash
   npm run dev
   ```

## الميزات المتقدمة

### الأمان
- **Row Level Security (RLS)**: سياسات أمان مبنية على الأدوار لحماية البيانات
- **Validations**: تحقق من صحة البيانات عند الإدخال
- **Audit Trails**: تتبع جميع العمليات للامتثال والسلامة

### الأداء
- **Caching Layer**: نظام ذاكرة تخزين مؤقت متطور لتقليل أوقات الاستجابة
- **Performance Monitoring**: مراقبة الأداء وتحليل الاستعلامات البطيئة
- **Optimized Indexes**: فهارس محسنة للاستعلامات الشائعة

### قاعدة البيانات
- **Triggers**: عمليات تحقق تلقائية وتحديثات العمل
- **Functions**: منطق عمل مركزي في قاعدة البيانات
- **Views**: عروض مخصصة لمراقبة الأداء
